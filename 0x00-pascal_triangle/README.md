Pascal’s Triangle is a triangular array of numbers named after the French mathematician Blaise Pascal, although it was studied by other mathematicians centuries before him in Persia, India, China, Germany, and Italy1.

The triangle starts with “1” at the top. Each number in the triangle is the sum of the two numbers directly above it. The rows and entries of Pascal’s Triangle are conventionally enumerated starting with row 0 at the top. The entries in each row are numbered from the left beginning with 0 and are usually staggered relative to the numbers in the adjacent rows1.

Here’s how it’s constructed:

In row 0 (the topmost row), there is a unique nonzero entry 1.
Each entry of each subsequent row is constructed by adding the number above and to the left with the number above and to the right, treating blank entries as 01.
Pascal’s Triangle has many applications in mathematics, including in calculations of combinations (binomial coefficients), algebra, and probability theory1. It also contains several interesting patterns:

The triangle is symmetrical.
The horizontal sums double each time.
Each line can also represent powers of 112.
It can be used to find squares of numbers2.
It can be used to generate the Fibonacci sequence2.
It can be used to calculate probabilities in coin tosses2.
For example, if you toss a coin three times, there is only one combination that will give three heads (HHH), but there are three that will give two heads and one tail (HHT, HTH, THH), also three that give one head and two tails (HTT, THT, TTH) and one for all Tails (TTT). This is represented by the pattern “1,3,3,1” in Pascal’s Triangle
